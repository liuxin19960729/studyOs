### FPU
```
了解
FPU 被称为x87芯片 ,x86也集成了FPU芯片,在里面被叫做X87部分


要想完成浮点数的功能也必须具备以下的一些条件
    1. 存储浮点数的空间
    2.操作浮点数的指令



FPU堆栈的其他的一些叫法
    FPU堆栈  x87堆栈
FPU指令集叫做x87指令集

```


```
mov  eax ， 1      //eax  传输需要请求cpuid功能码为1的参数
cpuid 
/*
cpuid
    作用是允许软件发现处理器的详细信息
    通过使用CPUID操作码，软件可以确定处理器的类型和特性支持

    执行cpuid之后 返回的值存在EAX里面

    ECX可以指定cpu要返回的信息

    cpuid 返回的数据宽度为两个byte也就是word(字)
*/
xor  rax ， rax   //初始化 rax寄存器
// xor 指令  作用为 1 xor 1 =0  0 xor 0=0   其他为1
bt  edx ， rax    
setc  al         
```
