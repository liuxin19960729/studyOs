
### 指针

初始化
```
char a ;
statci char b;
char *c;
上面申明了中变量他们初始化的区别是
    char a  基础类型的变量会在内存中划分一块存储区域，但是不是初始化数据
    static char b  静态变量会在堆栈内存里划分一块区域，会根据不同的类型进行初始化的参数 例  int  0 boolean false

    char * c 指针申明的时候不给他初始化的化，他是不会像前两个在内存里面进行开辟内存的

    注意：所以使用指针的时候一定要进行初始化。这样使用这个指针变量的时候才不会进行报错

```



数组和指针的关系

```
int  a[10]={0};//申明一个int类型的数组

a 就是 数组的第一个指针
*a=a[0] a指针变量的指向和数组的第一个元素是相等的

指针变量也是可以进行加减整数的
注意 :  这离为了安全运用指针加减整数是有一个条件的   对连续地址的数组进行加减，前提是不超过数组的范围
   范围   0<   >数组的长-1

   注意：当我们如果超过了范围的化编译器是不会报错的，但是这样我们的程序会非常危险，可能会错误修改有些值导致我们程序出错

   a+1 这个指针 1不是加一个byte大小,编译器会帮我们优化 内部会加上a+sizeof(int)*1  这个大小的b数量



当然指针加不但可以加减整数还能进行   指针-指针的操作

        条件:还是对同一个数组里面元素的指针进行操作
        
        int *p1 =&a[i];
        int *p2=&a[j];

        p1-p2==i-j

        注意 p1-p2 的值不是几个byte而是数组元素相差索引值

        编译器会进行操作  (p1-p2)/sizeof(int)
```
